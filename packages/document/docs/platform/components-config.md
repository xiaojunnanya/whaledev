# 组件配置项

## 通用

- `prompt`: 配置 prompt 可展示提示词

- ```
  // 当inputMode 的值等于 XXX 的时候不显示
  ignoreConfig: [
      'inputMode === Input.TextArea',
      'inputMode === Input.OTP',
  ],
  ```
